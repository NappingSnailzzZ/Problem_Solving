# [프로그래머스] 완주하지 못한 선수

> 프로그래머스 | Lv. 1(정답률 60%) | 해시 | C++

## 문제 요약
- 참가자 명단과 완주자 명단이 주어지고, 완주하지 못한 단 한 명을 찾기
- 동명이인 가능, 참가자 수 ≤ 100,000

## 풀이 접근
1. `unordered_map`에 참가자 이름별 카운트를 증가시키고, 완주자 이름별로 감소
2. 카운트가 0이 아닌 이름이 완주하지 못한 선수
3. 시간복잡도 분석: O(N) — unordered_map의 삽입/조회가 평균 O(1)

## 내 코드
```cpp
#include <string>
#include <vector>
#include <unordered_map>
using namespace std;
string solution(vector<string> participant, vector<string> completion) {
    unordered_map<string, int> m;
    for(auto& it : participant) {
        m[it]++;
    }
    for(auto& it : completion) {
        m[it]--;
    }
    for(auto& it : participant) {
        if(m[it] != 0)
            return it;
    }
}
```

## 리뷰
- 효율성을 위해 `map` → `unordered_map`으로 바꿈
- 그래도 효율성 테스트를 통과하지 못해 `auto it`에서 `auto& it`로 바꾸니 통과함
