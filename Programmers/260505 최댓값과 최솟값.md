# [프로그래머스] 최댓값과 최솟값

> 프로그래머스 | Lv. 2(정답률 80%) | 문자열, 자료구조 | C++

## 문제 요약
- 공백으로 구분된 정수들이 담긴 문자열에서 최솟값과 최댓값을 찾아 "(최소값) (최대값)" 형태로 반환
- 음수 포함 가능, 정수 2개 이상 보장

## 풀이 접근
1. 문자열을 한 글자씩 순회하면서 공백 또는 끝에 도달하면 숫자 문자열을 `stoi`로 변환
2. `set`의 자동 정렬을 활용 → `begin()`이 최솟값, `rbegin()`이 최댓값
3. `min_element`/`max_element`로도 가능하지만, 의도적으로 `set`을 사용하여 연습
4. 시간복잡도 분석: O(NlogN)(N: 숫자 개수, set 삽입 비용)

## 내 코드
```cpp
#include <string>
#include <vector>
#include <set>
using namespace std;
string solution(string s) {
    string answer = "";
    set<int> numbers;
    string s_num = "";
    for(int i=0; i<s.length(); i++) {
        if(s[i] != ' ')
            s_num += s[i];
        
        if(s[i] == ' ' || i == s.length()-1) {
            numbers.insert(stoi(s_num));
            s_num = "";
        }
    }
    answer += to_string(*numbers.begin()) + ' ' + to_string(*numbers.rbegin());
    return answer;
}
```

## 리뷰
- `set`은 삽입 시 자동 정렬되므로 최솟값/최댓값을 `begin()`/`rbegin()`으로 바로 접근할 수 있음
- 마지막 숫자는 뒤에 공백이 없으므로 `i == s.length()-1` 조건이 필요함.
- 원소가 하나인 경우에도 `begin() == rbegin()`이므로 정상 동작함
