# [프로그래머스] 124 나라의 숫자

> 프로그래머스 | Lv. 2(정답률 51%) | 수학, 진법 변환 | C++

## 문제 요약
- 1, 2, 4 세 개의 숫자만 사용하여 모든 자연수를 표현
- 0이 존재하지 않는 3진법 변형 체계
- 자연수 n을 124 나라의 숫자로 변환하여 반환

## 풀이 접근
1. 기본적으로 3으로 나누며 자릿수를 결정하는 3진법 변환과 유사
2. 나머지가 0인 경우 일반 3진법처럼 '0'을 두는 대신 '4'를 배치하고, 몫에서 1을 빼줌 (`n = n/3 - 1`)
3. 나머지가 1, 2인 경우는 일반 3진법과 동일하게 처리
4. 시간복잡도 분석: O(log₃ N)

## 내 코드
```cpp
#include <string>
#include <vector>
using namespace std;
string solution(int n) {
    string answer = "";
    while(n>0) {
        if(n%3 == 0) {
            answer = '4' + answer;
            n = n/3 -1;
        }
        else if(n%3 == 1) {
            answer = '1' + answer;
            n /= 3;
        }
        else if(n%3 == 2) {
            answer = '2' + answer;
            n /= 3;
        }
    }
    return answer;
}
```

## 리뷰
- 나머지 0일 때 `n = n/3 - 1` 처리가 중요. 0이 없는 진법 체계에서는 나머지가 0이면 가장 큰 숫자(여기선 4)를 배치하고 몫을 하나 줄여야 함.
- `else if`를 세 번 쓰는 대신, 분기를 하나로 줄일 수 있음
- 문자열 앞에 계속 붙이는 방식은 매번 문자열 복사가 발생해 큰 수라면 비효율적
