# [PCCP 기출문제] 1번 / 붕대 감기

> 프로그래머스 | Lv. 1 (정답률 45%) | 시뮬레이션 | C++

## 문제 요약
- 붕대 감기 기술: t초 연속 시전 시 초당 x 회복 + 추가 y 회복
- 몬스터 공격 시 기술 취소, 연속 성공 초기화, 해당 초에는 회복 불가
- 체력이 최대 체력을 초과할 수 없고, 0 이하가 되면 사망(-1 반환)
- 마지막 공격 직후 남은 체력을 반환

## 풀이 접근
1. 0초부터 마지막 공격 시간까지 1초 단위로 시뮬레이션하는 방식으로 접근
2. 공격 시간인지 아닌지를 `atk_check` 인덱스로 판별하여 분기 처리
3. 공격이 아닌 초에는 초당 회복 + 연속 성공 카운트 증가, t초 달성 시 추가 회복 후 카운트 초기화
4. 시간복잡도 분석: O(마지막 공격 시간) → 최대 O(1000)

## 내 코드
```cpp
#include <string>
#include <vector>
using namespace std;
int solution(vector<int> bandage, int health, vector<vector<int>> attacks) {
    int health2 = health; // 현재 체력
    int cnt = 0; // 연속 성공
    int atk_check = 0;
    for(int i=0; i<=attacks[attacks.size()-1][0]; i++) {
        if(i == attacks[atk_check][0]) {
            health2 -= attacks[atk_check][1];
            cnt = 0;
            atk_check++;            
        }
        else {
            health2 += bandage[1];
            cnt++;
            
            if(cnt == bandage[0]) {
                cnt = 0;
                health2 += bandage[2];
            }
        }
        if(health2<=0)
            return -1;
        if(health2>health)
            health2 = health;
    }
    return health2;
}
```

## 리뷰
- 시뮬레이션 문제는 시간 흐름을 for문으로 돌리면서 조건 분기하는 패턴이 기본이라는 걸 확인
- 최대 체력 제한은 회복과 공격 모두 처리한 뒤 마지막에 해주는 게 순서상 안전
