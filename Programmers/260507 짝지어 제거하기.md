# [프로그래머스] 짝지어 제거하기

> 프로그래머스 | Lv. 2(정답률 75%) | 스택 | C++

## 문제 요약
- 문자열에서 같은 알파벳이 2개 붙어 있으면 제거하고 앞뒤를 이어 붙이는 과정을 반복
- 문자열을 모두 제거할 수 있으면 1, 아니면 0 반환
- 문자열 길이 ≤ 1,000,000

## 풀이 접근
1. 스택을 활용: 현재 문자가 스택 top과 같으면 pop(짝 제거), 다르면 push
2. 실제로 문자열을 매번 잘라 붙이면 O(N²)이지만, 스택을 쓰면 한 번의 순회로 해결 가능
3. 순회 완료 후 스택이 비어 있으면 모두 제거된 것이므로 1 반환
4. 시간복잡도 분석: O(N), 공간복잡도 O(N)

## 내 코드
```cpp
#include <iostream>
#include <string>
#include <stack>
using namespace std;
int solution(string s) {
    stack<char> st;
    for(int i=0; i<s.length(); i++) {
        if(st.empty())
            st.push(s[i]);
        else if(st.top() == s[i])
            st.pop();
        else
            st.push(s[i]);
    }
    
    if(st.size() == 0)
        return 1;
    else
        return 0;
}
```

## 리뷰
- C++ STL `stack` 처음 사용. `push()`, `pop()`, `top()`, `empty()`, `size()` 활용
- "연속된 짝을 제거하고 이어 붙인다" 패턴이 나오면 스택을 떠올리는 게 정석.
- 스택이 비어있으면 `top()` 호출 시 오류가 발생하므로 `st.empty()` 체크를 먼저 함
- `st.empty()`와 `st.size() == 0`를 모두 사용했는데 `st.empty()`를 쓰는 게 더 일반적
