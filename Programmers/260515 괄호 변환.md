# [프로그래머스] 괄호 변환

> 프로그래머스 | Lv. 2(정답률 49%) | 2020 KAKAO BLIND RECRUITMENT | 재귀, 문자열, 구현 | C++

## 문제 요약
- 균형잡힌 괄호 문자열을 주어진 알고리즘에 따라 올바른 괄호 문자열로 변환
- u는 더 이상 분리할 수 없는 최소 균형잡힌 문자열, v는 나머지
- u가 올바르면 그대로 두고, 아니면 뒤집기 등의 가공 후 재조합

## 풀이 접근
1. 올바른 괄호 판별 함수(isRight)와 재귀 변환 함수(recursive_answer)를 분리하여 구현
2. u/v 분리: 왼쪽부터 '(' +1, ')' -1로 카운트하여 0이 되는 최초 지점에서 분할
3. u가 올바르면 u + 재귀(v)로 반환
4. u가 올바르지 않으면 괄호 방향을 뒤집은 뒤 "(" + 재귀(v) + ")" + u의 앞뒤 제거본으로 조합
5. 시간복잡도 분석: 각 재귀 단계에서 O(N) 작업, 최대 O(N/2)번 재귀 → O(N²)

## 내 코드
```cpp
#include <string>
#include <vector>
#include <stack>
using namespace std;
bool isRight(string str) {
    stack<char> st;
    for(auto& it : str) {
        if(st.empty() && it == ')')
            return false;
        else if(it == '(')
            st.push(it);
        else
            st.pop();
    }
    return true;
}
string recursive_answer(string str) {
    if(str == "")
        return "";
    string u, v;
    int i = 0;
    int cnt = 0;
    for(auto& it : str) {
        if(it == '(')
            cnt++;
        else
            cnt--;
        i++;
        if(cnt == 0)
            break;
    }
    u = str.substr(0, i);
    v = str.substr(i, str.length()-i);
    
    if(isRight(u)) {
        return u + recursive_answer(v);
    }
    else {
        for(auto& it : u) {
            if(it == '(')
                it = ')';
            else
                it = '(';
        }
        return "(" + recursive_answer(v) + ")" + u.substr(1,u.length()-2);
    }
}
string solution(string p) {
    return recursive_answer(p);
}
```

## 리뷰
- isRight 함수에서 스택을 사용했지만 단순한 카운터로도 충분함
- 함수의 매개변수를 `const string&`로 받으면 불필요한 복사를 줄일 수 있음
- 문제에서 알고리즘을 그대로 제시해주기에 정확히 구현하는게 중요
