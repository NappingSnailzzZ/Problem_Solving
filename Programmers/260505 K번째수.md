# [프로그래머스] K번째수

> 프로그래머스 | Lv. 1(정답률 73%) | 정렬 | C++

## 문제 요약
- 배열에서 i~j번째 구간을 잘라 정렬한 뒤 k번째 수를 구하는 연산을 commands마다 반복
- array 길이 ≤ 100, commands 길이 ≤ 50

## 풀이 접근
1. commands를 순회하면서 매번 부분 배열을 추출 → 정렬 → k번째 원소 선택
2. vector의 iterator 범위 생성자로 subvector를 만들 수 있음
3. 시간복잡도 분석: O(M * NlogN) (M: commands 수, N: array 길이)

## 내 코드
```cpp
#include <string>
#include <vector>
#include <algorithm>
using namespace std;
vector<int> solution(vector<int> array, vector<vector<int>> commands) {
    vector<int> answer;
    for(int i=0; i<commands.size(); i++) {
        vector<int> subvector(array.begin()+commands[i][0]-1, array.begin()+commands[i][1]);
        sort(subvector.begin(), subvector.end());
        answer.push_back(subvector[commands[i][2]-1]);
    }
    return answer;
}
```

## 리뷰
- vector의 iterator 범위 생성자를 활용하면 부분 벡터 추출이 가능하다는 점을 활용
- `nth_element`를 쓰면 전체 정렬 및 부분 벡터 추출 없이 해결 가능
