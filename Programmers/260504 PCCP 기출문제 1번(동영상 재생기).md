# [프로그래머스 PCCP 기출문제 1번] 동영상 재생기

> 프로그래머스 | Lv. 1(정답률 42%) | 구현, 시뮬레이션 | C++

## 문제 요약
- 동영상 재생기에 "prev"(10초 전), "next"(10초 후), 오프닝 건너뛰기 3가지 기능이 있음
- prev/next 시 영상 범위(0 ~ video_len)를 벗어나면 경계값으로 클램핑
- 현재 위치가 오프닝 구간(`op_start ≤ pos ≤ op_end`)이면 자동으로 op_end로 이동
- commands를 순서대로 수행한 뒤 최종 위치를 "mm:ss" 형식으로 반환

## 풀이 접근
1. "mm:ss" 문자열을 `분*60 + 초`로 변환해서 정수 하나로 관리 → 비교/연산이 편해짐
2. 명령 수행 전에도 오프닝 체크를 한 번 하고, 각 명령 수행 후마다 경계 클램핑 → 오프닝 체크 순서로 처리
3. 최종 위치를 다시 분/초로 나눠서 포맷팅
4. 시간복잡도: O(N)(N = commands 길이, 최대 100)

## 내 코드
```cpp
#include <string>
#include <vector>
using namespace std;
string solution(string video_len, string pos, string op_start, string op_end, vector<string> commands) {
    string answer = "";
    int video_len2 = stoi(video_len.substr(0,2)) * 60 + stoi(video_len.substr(3,2));
    int pos2 = stoi(pos.substr(0,2)) * 60 + stoi(pos.substr(3,2));
    int op_start2 = stoi(op_start.substr(0,2)) * 60 + stoi(op_start.substr(3,2));
    int op_end2 = stoi(op_end.substr(0,2)) * 60 + stoi(op_end.substr(3,2));
    
    if(pos2>=op_start2 && pos2<op_end2)
        pos2 = op_end2;
    
    for(int i=0; i<commands.size(); i++) {
        if(commands[i] == "next")
            pos2 += 10;
        else if(commands[i] == "prev")
            pos2 -= 10;
        
        if(pos2<0)
            pos2 = 0;
        if(pos2>video_len2)
            pos2 = video_len2;
        if(pos2>=op_start2 && pos2<op_end2)
            pos2 = op_end2;
    }
    int min = pos2/60;
    int sec = pos2%60;
    string min_str = to_string(min);
    string sec_str = to_string(sec);
    if(min<10)
        min_str = '0' + min_str;
    if(sec<10)
        sec_str = '0' + sec_str;
    
    answer = answer + min_str + ':' + sec_str;
    return answer;
}
```

## 리뷰
- 시간 문자열 → 초 단위 정수 변환은 시간 관련 문제에서 효과가 큼
- 오프닝 체크 조건에서 `pos2 < op_end2`를 사용했는데, 문제 원문은 `op_start ≤ pos ≤ op_end`임. 명세와 같은 조건을 구현하는게 안전함
