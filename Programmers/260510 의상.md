[프로그래머스] 의상

> 프로그래머스 | Lv. 2(정답률 68%) | 해시, 조합론 | C++

## 문제 요약
- 종류별로 최대 1개의 의상만 착용 가능
- 하루에 최소 1개는 착용해야 함
- 서로 다른 옷 조합의 수를 구하기

## 풀이 접근
1. 종류별 의상 개수를 해시맵으로 카운팅
2. 각 종류마다 "안 입는 경우"를 포함해 (개수 + 1)가지 선택지가 생김
3. 모든 종류의 선택지를 곱하면 전체 경우의 수 → 여기서 아무것도 안 입는 경우 1을 빼면 정답
4. 시간복잡도 분석: O(N) (N = 의상 수)

## 내 코드
```cpp
#include <string>
#include <vector>
#include <unordered_map>
using namespace std;
int solution(vector<vector<string>> clothes) {
    int answer = 1;
    unordered_map<string, int> count;
    for(auto& it : clothes)
        count[it[1]]++;
    
    for(auto& it : count)
        answer *= it.second + 1;
    
    return answer - 1;
}
```

## 리뷰
- 곱의 법칙을 활용한 문제. 각 종류를 독립적인 선택으로 보고 "안 입는다"를 하나의 선택지로 추가하여 곱함
- 전부 안 입는 경우는 제외해야 하므로 1 빼기
- 이름은 구분할 필요 없고 종류별 개수만 알면 되니 해시맵으로 카운팅
