# [프로그래머스] 2022 KAKAO BLIND RECRUITMENT k진수에서 소수 개수 구하기

> 프로그래머스 | Lv. 2(정답률 63%) | 문자열, 소수 판별 | C++

## 문제 요약
- 양의 정수 n을 k진수로 변환한 뒤, 0을 구분자로 잘라서 나오는 숫자들 중 소수의 개수를 구하기
- P는 각 자릿수에 0을 포함하지 않는 소수
- 소수 판별은 10진법 기준

## 풀이 접근
1. 나머지 연산으로 n을 k진수 문자열로 변환
2. 0이 하나도 없는 경우를 먼저 처리 → 문자열 전체가 하나의 후보
3. 그 외에는 0을 구분자로 잘라가며 각각 소수 판별
4. 소수 판별은 별도 함수로 분리
5. 시간복잡도 분석: 변환 O(log n), 각 토큰에 대해 소수 판별 O(√P)

## 내 코드
```cpp
#include <string>
#include <vector>
#include <cmath>
using namespace std;
bool isPrime(long long n) {
    if(n == 1)
        return false;
    for(int i=2; i<=sqrt(n); i++) {
        if(n%i == 0)
            return false;
    }
    return true;
}
int solution(int n, int k) {
    int answer = 0;
    string num = "";
    while(n>0) {
        num = to_string(n%k) + num;
        n /= k;
    }
    int cnt = 0;
    int pointer = 0;
    bool zero = false;
    for(int i=0; i<num.length(); i++) {
        if(num[i] == '0')
            zero = true;
    }
    if(!zero && isPrime(stoll(num)))
        return 1;
        
    for(int i=0; i<num.length(); i++) {
        if(num[i] != '0') {
            cnt++;
        }
        else {
            if(cnt != 0 && isPrime(stoll(num.substr(pointer, cnt))))
                answer++;
            cnt = 0;
            pointer = i+1;
        }
    }
    if(cnt != 0 && isPrime(stoll(num.substr(pointer, num.length()-pointer))))
        answer++;
    
    return answer;
}
```

## 리뷰
- stoi → stoll 변경으로 오버플로우 해결
- 소수 판별 루프 변수 `i`가 int인데, sqrt(n)이 매우 클 경우 비교 시 암묵적 형변환 이슈가 생길 수 있음. `long long i`로 맞추거나 `i * i <= n` 패턴을 쓰면 sqrt의 부동소수점 오차도 피할 수 있음
