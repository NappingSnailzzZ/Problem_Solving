# [프로그래머스] 2 x n 타일링

> 프로그래머스 | Lv. 2(정답률 59%) | DP | C++

## 문제 요약
- 2×1 타일로 2×n 바닥을 채우는 경우의 수 구하기
- n ≤ 60,000이며 결과를 1,000,000,007로 나눈 나머지를 반환

## 풀이 접근
1. n번째 칸을 채우는 방법은 세로 타일 1개(n-1에서 옴) 또는 가로 타일 2개(n-2에서 옴) → 피보나치 점화식 `dp[i] = dp[i-1] + dp[i-2]`
2. 초기값 answer[1]=1, answer[2]=2로 설정
3. 매 단계마다 MOD 연산을 적용하여 오버플로우 방지
4. 시간복잡도 분석: O(N), 공간복잡도 O(N)

## 내 코드
```cpp
#include <string>
#include <vector>
using namespace std;
int solution(int n) {
    vector<int> answer = {0, 1, 2};
    for(int i=3; i<=n; i++)
        answer.push_back((answer[i-2]+answer[i-1])%1000000007);
    
    return answer[n];
}
```

## 리뷰
- 2×n 타일링은 대표적인 피보나치 변형 DP 문제. 점화식만 세우면 구현은 간단
- 직전 두 값만 필요하므로 변수 2개로 줄이면 O(1) 공간으로 개선 가능
- `int` 범위 주의: 안전하게 `long long`을 사용할 수 있음
