# [Codetree] 진수 to 진수

> Codetree | Easy(정답률 74%) | 수학, 진법 변환 | C++

## 문제 요약
- A진수로 표현된 수 N을 B진수로 변환하여 출력
- 2 ≤ A, B ≤ 9, N의 자릿수 ≤ 9

## 풀이 접근
1. A진수 → 10진수: 문자열의 각 자릿수를 순회하며 `target = target * A + 현재 자릿수`로 변환
2. 10진수 → B진수: B로 나누면서 나머지를 벡터에 저장, 역순 출력
3. 10진수를 중간 다리로 활용하는 표준적인 진법 변환 흐름
4. 시간복잡도 분석: O(자릿수)

## 내 코드
```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int main() {
    int a, b, n;
    cin >> a >> b >> n;
    int target = 0;
    string s = to_string(n);
    for(auto& it : s)
        target = target*a + (int)(it - '0');
    vector<int> v;
    while(target>0) {
        v.push_back(target%b);
        target /= b;
    }
    for(int i=v.size()-1; i>=0; i--)
        cout << v[i];
    return 0;
}
```

## 리뷰
- A진수 → 10진수 변환에서 (`target = target * a + digit`)를 사용해 한 번의 순회로 처리됨
- N을 처음부터 string으로 입력받으면 변환 과정이 하나 줄어듦
- 역순 출력을 위해 string에 앞으로 붙이는 방식도 가능
