# [Codetree] 그 요일은

> Codetree | Medium(정답률 35%) | 수학, 구현 | C++

## 문제 요약
- 2024년(윤년) m1월 d1일이 월요일일 때, m1월 d1일부터 m2월 d2일까지 특정 요일 A가 몇 번 등장하는지 구하기
- m2/d2가 m1/d1보다 앞선 경우는 없음

## 풀이 접근
1. m1월부터 m2월 전까지의 월별 일수를 합산하고, d2-d1을 더해 두 날짜 사이의 gap(일수 차이) 계산
2. m1/d1이 월요일이므로, 요일 A의 오프셋에 따라 `(gap + 7 - offset) / 7`로 등장 횟수를 산출
3. Mon=0, Tue=1, ..., Sun=6 오프셋 기준으로 if-else 분기
4. 시간복잡도 분석: O(1)

## 내 코드
```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int main() {
    int m1, d1, m2, d2;
    string a;
    cin >> m1 >> d1 >> m2 >> d2 >> a;
    vector<int> days = {0, 31, 29, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
    int gap = 0;
    for(int i=m1; i<m2; i++)
        gap += days[i];
    gap += (d2 - d1);
    if(a == "Mon")
        cout << (gap+7) / 7;
    else if(a == "Tue")
        cout << (gap+6) / 7;
    else if(a == "Wed")
        cout << (gap+5) / 7;
    else if(a == "Thu")
        cout << (gap+4) / 7;
    else if(a == "Fri")
        cout << (gap+3) / 7;
    else if(a == "Sat")
        cout << (gap+2) / 7;
    else
        cout << (gap+1) / 7;
    return 0;
}
```

## 리뷰
- if-else 7개를 map과 배열로 대체하면 훨씬 간결하게 구현 가능
