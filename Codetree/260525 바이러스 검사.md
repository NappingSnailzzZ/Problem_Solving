# [Codetree] 바이러스 검사

> Codetree | 삼성 2015 하반기 1번(정답률 37%) | 수학, 그리디 | C++

## 문제 요약
- n개의 식당, 각 식당마다 고객 수가 주어짐
- 검사팀장은 최대 a명, 검사팀원은 최대 b명 검사 가능
- 각 식당에 팀장 1명은 반드시 필요, 팀원은 0명 이상
- 모든 식당의 고객을 검사하기 위한 최소 검사자 수를 구함

## 풀이 접근
1. 각 식당마다 팀장 1명은 무조건 배정 → 팀장이 a명을 먼저 처리
2. 남은 고객이 있으면 팀원으로 커버 → 올림 나눗셈으로 필요한 팀원 수 계산
3. 핵심 식: 남은 인원 = max(0, 고객수 - a), 팀원 수 = ceil(남은 인원 / b) = `(it - a - 1) / b + 1`
4. 각 식당 독립적으로 계산 가능 → 총합이 답
5. 시간복잡도: O(n)

## 내 코드
```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n, a, b;
    cin >> n;
    vector<int> c(n);
    for(int i=0; i<n; i++)
        cin >> c[i];
    cin >> a >> b;
    long long answer = 0;
    for(auto& it : c) {
        if(a>=it) {
            answer++;
            continue;
        }
        else {
            answer += (it-a-1)/b + 2;
        }
    }
    cout << answer;
    return 0;
}
```

## 리뷰
- 올림 나눗셈 `ceil(x/y) = (x-1)/y + 1`이 핵심
- answer는 int 범위를 초과할 수 있으므로 long long으로 선언한 점이 좋음
