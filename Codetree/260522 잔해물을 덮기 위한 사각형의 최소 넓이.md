# [Codetree] 잔해물을 덮기 위한 사각형의 최소 넓이

> Codetree | Medium(정답률 33%) | 기하, 케이스 분류 | C++

## 문제 요약
- 첫 번째 직사각형 위에 두 번째 직사각형을 덮었을 때, 남은 첫 번째 직사각형의 잔해물을 덮는 최소 직사각형 넓이를 구함
- 좌측하단 (x1, y1), 우측상단 (x2, y2) 형식으로 두 직사각형이 주어짐
- 좌표 범위: -1000 ~ 1000

## 풀이 접근
1. 두 번째 직사각형이 첫 번째를 어떻게 가리는지에 따라 케이스를 분류
2. 핵심 관찰: 잔해물의 바운딩 박스를 줄일 수 있는 경우는, 한 축 방향이 완전히 덮일 때만 다른 축을 축소할 수 있음
   - 완전히 덮힘 → 넓이 0
   - x축 완전 커버 → y축만 축소
   - y축 완전 커버 → x축만 축소
   - 둘 다 부분 커버 → 축소 불가, 원래 넓이 유지
3. 시간복잡도: O(1)

## 내 코드
```cpp
#include <iostream>
using namespace std;

int main() {
    int x1, y1, x2, y2, x3, y3, x4, y4;
    cin >> x1 >> y1 >> x2 >> y2 >> x3 >> y3 >> x4 >> y4;
    int a = x2-x1, b = y2-y1;
    if((x1<x3 || x2>x4) && (y1<y3 || y2>y4)) {
        cout << a*b;
        return 0;
    }
    else if((x1>=x3 && x2<=x4) && (y1>=y3 && y2<=y4)) {
        cout << 0;
        return 0;
    }
    else if((x1>=x3 && x2<=x4) && (y1<y3 || y2>y4)) {
        if((y1<y3) && (y2>y4)) {
            cout << a*b;
            return 0;
        }
        else if((y1>=y3)) {
            b = y2 - y4;
            cout << a*b;
            return 0;
        }
        else {
            b = y3 - y1;
            cout << a*b;
            return 0;
        }
    }
    else if((x1<x3 || x2>x4) && (y1>=y3 && y2<=y4)) {
        if((x1<x3) && (x2>x4)) {
            cout << a*b;
            return 0;
        }
        else if((x1>=x3)) {
            a = x2 - x4;
            cout << a*b;
            return 0;
        }
        else {
            a = x3 - x1;
            cout << a*b;
            return 0;
        }
    }
    cout << a*b;
    return 0;
}
```

## 리뷰
- 기하 문제에서 케이스 분류는 빠짐없이 정리하는 게 중요
- 원래 크기인 경우: 겹침이 아예 없음, 부분 겹침일 때 잔해물이 L자 형태, 관통하는 겹침 형태
- x축/y축 대칭 구조이므로 한쪽 로직을 짠 뒤 축만 바꿔서 복사하면 실수를 줄일 수 있음
