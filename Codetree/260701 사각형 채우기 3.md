# [Codetree] 사각형 채우기 3

> Codetree | Medium(정답률 31%) | DP | C++

## 문제 요약
- 2×N 크기의 사각형을 1×2, 2×1, 1×1 타일로 채우는 방법의 수를 구함
- 결과를 10⁹+7로 나눈 나머지를 출력
- 1 ≤ N ≤ 1000

## 풀이 접근
1. dp[i] = 2×N에서 i열까지 채우는 방법의 수
2. 점화식 구성:
   - `dp[i-1] × 2`: 마지막 1열에 1×1 하나를 위 또는 아래에 놓는 경우
   - `dp[i-2] × 3`: 마지막 2열을 채우는 고유 패턴 3가지 (2×1 세로 두 개 / 1×2 가로 두 개 / 1×1 네 개... 가 아니라, 이전 항에서 이미 세지 않은 새로운 배치 3종)
   - `dp[j] × 2` (j = 0 ~ i-3): 3열 이상에 걸치는 고유 패턴이 각각 2가지씩 존재 (위아래 대칭)
3. 시간복잡도: O(N²)

## 내 코드
```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n;
    cin >> n;
    if(n==1) {
        cout << 2;
        return 0;
    }
    if(n==2) {
        cout << 7;
        return 0;
    }
    vector<long long> v(n+1);
    v[0] = 1;
    v[1] = 2;
    v[2] = 7;
    for(int i=3; i<=n; i++) {
        for(int j=i-3; j>=0; j--)
            v[i] = (v[i] + v[j]*2) % 1000000007;
        v[i] = (v[i] + v[i-1]*2 + v[i-2]*3) % 1000000007;
    }
    cout << v[n];
    return 0;
}
```

## 리뷰
- 2×N 타일링 DP의 확장형. 1×1 타일이 추가되면서 3열 이상에 걸치는 패턴이 생기고, 이를 누적합으로 처리해야 하는 점이 핵심
- 내부 for문으로 O(N²)이지만, 누적합 변수를 하나 두면 O(N)으로 최적화 가능
