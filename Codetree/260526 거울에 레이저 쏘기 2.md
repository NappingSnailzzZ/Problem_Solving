# [Codetree] 거울에 레이저 쏘기 2

> Codetree | Medium(정답률 45%) | 시뮬레이션 | C++

## 문제 요약
- N×N 격자에 각 칸마다 `\` 또는 `/` 거울이 배치
- 격자 외곽 4N개의 위치 중 K번 위치에서 레이저를 쏨
- K는 위에서 아래 방향으로 1행 1열 칸부터 시계 방향으로 번호를 매긴 값
- 레이저가 거울에 튕기는 횟수를 출력

## 풀이 접근
1. K값으로 레이저 시작 위치(x, y)와 초기 방향(d)을 결정 → 4방향 분기
   - K ≤ N: 위쪽 변 → 아래 방향(0)
   - N < K ≤ 2N: 오른쪽 변 → 왼쪽 방향(1)
   - 2N < K ≤ 3N: 아래쪽 변 → 위 방향(2)
   - 3N < K ≤ 4N: 왼쪽 변 → 오른쪽 방향(3)
2. 거울별 방향 전환 규칙:
   - `/`: ↓↔← , ↑↔→
   - `\`: ↓↔→ , ↑↔←
3. 격자 안에 있는 동안 반복: 거울로 방향 전환 → 이동 → 카운트 증가
4. 시간복잡도: 최악 O(N²) (격자를 대각선으로 지그재그할 경우)

## 내 코드
```cpp
#include <iostream>
#include <vector>
using namespace std;

bool inRange(int n, int x, int y) {
    return (x>=0 && x<n && y>=0 && y<n);
}

int main() {
    int n, k, x, y, answer = 0;
    int d; // ↓:0, ←:1, ↑:2, →:3
    cin >> n;
    vector<vector<char>> v(n, vector<char>(n));
    vector<int> dx = {1, 0, -1, 0};
    vector<int> dy = {0, -1, 0, 1};
    for(int i=0; i<n; i++)
        for(int j=0; j<n; j++)
            cin >> v[i][j];
    cin >> k;
    if(k<=n) {
        d = 0;
        x = 0;
        y = k-1;
    }
    else if(k>n && k<=n*2) {
        d = 1;
        x = k-n-1;
        y = n-1;
    }
    else if(k>=n*2 && k<=n*3) {
        d = 2;
        x = n-1;
        y = n*3 - k;
    }
    else {
        d = 3;
        x = n*4 - k;
        y = 0;
    }
    while(inRange(n, x, y)) {
        if(v[x][y] == '/') {
            if(d == 0)
                d = 1;
            else if(d == 1)
                d = 0;
            else if(d == 2)
                d = 3;
            else
                d = 2;
        }
        else  {
            if(d == 0)
                d = 3;
            else if(d == 1)
                d = 2;
            else if(d == 2)
                d = 1;
            else
                d = 0;
        }
        x += dx[d];
        y += dy[d];
        answer++;
    }
    cout << answer;
    return 0;
}
```

## 리뷰
- 실수하기 쉬운 부분이 많으므로 예제로 반드시 검증할 것
- 방향 전환 규칙을 if-else로 직접 나열했는데, 배열 매핑으로 바꾸면 코드가 훨씬 간결해짐
- "먼저 방향 전환 → 그 다음 이동"하는 순서가 중요
