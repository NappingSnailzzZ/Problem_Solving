# [Codetree] 요일 맞추기

> Codetree | Hard (정답률 44%) | 수학, 구현 | C++

## 문제 요약
- 2011년 m1월 d1일이 월요일일 때, 2011년 m2월 d2일의 요일을 구하기
- 2011년은 윤년이 아님 (2월 28일)

## 풀이 접근
1. 두 날짜를 각각 1월 1일 기준 총 일수로 변환
2. 두 날짜의 차이(gap)를 구한 뒤 7로 나눈 나머지로 요일 결정
3. gap이 양수인 경우와 음수인 경우를 나눠서 요일 매핑을 별도로 처리
4. 시간복잡도 분석: O(1) (월 수가 최대 12이므로 상수)

## 내 코드
```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int m1, d1, m2, d2;
    cin >> m1 >> d1 >> m2 >> d2;
    vector<int> days = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
    int day1 = 0;
    int day2 = 0;
    for(int i=1; i<m1; i++)
        day1 += days[i];
    day1 += d1;
    for(int i=1; i<m2; i++)
        day2 += days[i];
    day2 += d2;
    int gap = day2 - day1;

    if(gap >= 0) {
    if(gap%7 == 0)
        cout << "Mon";
    else if(gap%7 == 1)
        cout << "Tue";
    else if(gap%7 == 2)
        cout << "Wed";
    else if(gap%7 == 3)
        cout << "Thu";
    else if(gap%7 == 4)
        cout << "Fri";
    else if(gap%7 == 5)
        cout << "Sat";
    else if(gap%7 == 6)
        cout << "Sun";
    }
    else {
        if(-1*gap%7 == 0)
        cout << "Mon";
    else if(-1*gap%7 == 1)
        cout << "Sun";
    else if(-1*gap%7 == 2)
        cout << "Sat";
    else if(-1*gap%7 == 3)
        cout << "Fri";
    else if(-1*gap%7 == 4)
        cout << "Thu";
    else if(-1*gap%7 == 5)
        cout << "Wed";
    else if(-1*gap%7 == 6)
        cout << "Tue";
    }
    return 0;
}
```

## 리뷰
- days 배열 인덱스 0에 더미를 넣어서 n월=days[n]으로 맞춤
- `((gap % 7) + 7) % 7`를 사용하면 양수/음수 분기 필요 없음
- 요일 문자열을 배열을 이용하여 간결하게 표현가능
